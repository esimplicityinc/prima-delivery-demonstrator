# Bounded Contexts

## Overview

Bounded contexts define clear boundaries within the domain where a specific model applies.

---

## Adding New Contexts

When defining a new bounded context, include:
1. Clear purpose and responsibilities
2. Ubiquitous language specific to the context
3. Key aggregates and entities
4. Domain events published/consumed
5. Relationships with other contexts
